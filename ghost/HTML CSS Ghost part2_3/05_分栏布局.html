开启中俄关系

要说到钢铁产量，我国是毫无疑问的世界NO.1，占据了全球的半壁江山。不过，钢铁产品种类繁多，生产技术难度也大不相同，有一些领域，我国也曾面临“卡脖子”的问题。

但值得庆幸的是，随着我国科技实力的增强，已经有多项“卡脖子”技术被逐一攻破，其中有一项钢铁生产技术，我国不仅打破了日本、德国的长期垄断，还做到了世界领先水平，连美国都要排队购买！那么，到底是什么钢铁技术如此厉害？这就是接下来为各位讲到的“手撕钢”。

生产这种产品并不需要用到专用的特种钢材，用普通的不锈钢就能轧制而成，但正因为要求轧制得非常薄，工艺难度极大。因此，在过去，中国根本生产不出这种手撕钢，这种生产技术只掌握在日本和德国少数几家公司，我国完全依靠进口。

然而，手撕钢的价格是极其昂贵的，目前一片A4纸大小的手撕钢，市场价格就达到400元，可以说是论“克”卖，每克能卖到五六元，比白银价格还贵。当时我国每年的进口总额就超百亿元，而且还被限制只能进口厚度在0.03毫米以上的材料。

想必各位会疑惑，价格如此昂贵的手撕钢，到底有什么用？其实，用途多着呢！手撕钢属于不锈钢板带领域中的高端产品，因为具有优异的强度、精度、表面光洁等性能，同时还有强大的耐腐蚀、抗氧化、耐热等特点，被广泛应用于航空航天、电子、新能源、军工核电等领域。

以近年来比较火的折叠屏手机为例，它使用的柔性屏钢就是“手撕钢”的一个新品种，这种柔性屏钢可以承受20万次的折叠不变形、不断裂，如果消费者一天折叠手机屏幕50次，它的使用寿命可达到10年左右。
