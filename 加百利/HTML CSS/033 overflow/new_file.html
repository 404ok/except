大概讲一下杰玛的回忆。他从小住在贫民窟，和自己的哥哥舍姆利，家里很穷，但有一个爱自己的妈妈，兄弟俩爱玩，哥哥比较爱钱，弟弟有自己的坚持，就比如有一次，弟弟拿到了一个自己特别喜欢的明星的签名照，他特别开心，但是哥哥为了钱，把弟弟的签名照卖了。后来，在一次种族冲突中，妈妈被别人杀死了，兄弟俩只能相依为命，那天晚上下着大雨，外面有一个女孩，在淋雨，无家可归，杰玛让女孩到自己破破烂烂的家里躲雨，女孩名字叫拉提卡。三个人相依为命，后来三个人被抓小孩的抓了，开始对他们还行，但是后面哥哥发现他们会让唱歌唱的好的人变成瞎子，哥哥为了救弟弟，逃走了，但是拉提卡没有逃走，哥哥没有救她。兄弟俩在火车上生活了一段时间，后来在泰姬陵做旅游生意赚了很多钱。两个人回到了孟买，因为杰玛想要找到拉提卡。找到了，在逃跑过程中，哥哥为了未来活下去，杀了那个抓小孩的头头，哥哥投靠了抓小孩的敌人，在抓捕过程中，哥哥把拉提卡带走了，但是和弟弟分开了，后来找到了哥哥和拉提卡，拉提卡已经被迫成为了哥哥头头的情人。两人决定私奔，但是哥哥带入把拉提卡抓回去了，杰玛弟弟找不到他们了，他们搬走了，很伤心，决定参加拉提卡经常看的电视，答题百万富翁。后面题目都答对了，哥哥洗心革面救了拉提卡，哥哥死了，杰玛和拉提卡两人终于在一起了。 画面拍摄效果：里面有很多追逐画面，虽然可能有一些晃动，但是很连贯，节奏感也很好，还有里面对于光影的处理，很好看 剪辑点：有的是通过灯光的暗亮来转场，还有大动作（eg:警察打杰玛巴掌，老师打兄弟俩），局部细节（eg:警察的腿部特写再到全景）来实现 小孩追赶，展示贫民窟的全貌，后退式蒙太奇 转身，然后后退式蒙太奇，通过全景转
